## ⚙️ How to Run Locally

**🔹 1. Clone the Repository**
```bash
git clone https://github.com/hemaxplore/simple-portfolio.git
cd simple-portfolio
```

**🔹 2. Open Directly in Browser**
```bash
# Option A: Double-click index.html
# Option B: Use VS Code Live Server extension
```

**🔹 3. Configure EmailJS (for contact form)**
- Sign up at [emailjs.com](https://www.emailjs.com/)
- Replace your **Service ID**, **Template ID**, and **Public Key** in `index.html`:
```javascript
emailjs.init("YOUR_PUBLIC_KEY");
// In the form handler: emailjs.sendForm("SERVICE_ID", "TEMPLATE_ID", form)
```

> ✅ No Node.js, Python, or server setup required — works instantly in any browser!
